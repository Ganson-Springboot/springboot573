### 5.2.1管理员模块实现
管理员进入主页面，主要功能包括对个人中心，学生管理，教师管理，课程分类管理，班级管理，课程信息管理，学习课程管理，作业通知管理，学生作业管理，批改作业管理，留言板，在线讨论，系统管理等进行操作。管理员主页面如图5-7所示：

![a44be79ba36a94af90a097457cd051d](/md/blog.013.jpeg)

图5-7 管理员主界面
